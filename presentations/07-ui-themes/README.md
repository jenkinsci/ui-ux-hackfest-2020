# How to use and create Jenkins themes? + Dark Mode

Presented by
[Tobias Gruetzmacher](https://github.com/tobix),
[Oleg Nenashev](https://www.jenkins.io/blog/authors/oleg_nenashev/), and
[Tim Jacomb](https://www.jenkins.io/blog/authors/timja/).

In this presentation we will show how to use custom Web UI themes in Jenkins.
We will take a look at the Simple Theme Plugin and available themes,
and then we will configure our instance to use one of the themes.
Then we will try to create a custom theme for our instance.

Agenda:

* Introduction to the Simple Theme Plugin
* How to do a small changes to the UI?
* How to do a bigger theme or to create a new one?
* Jenkins UI/UX Hackfest: Working on a Jenkins Dark Theme together
